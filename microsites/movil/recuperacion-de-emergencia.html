“Título de página”

Si durante una actualización de software se produce algún fallo o error y el teléfono no arranca o se queda en el logo, puedes intentar una recuperación de emergencia para volver a instalar el software y recuperar el terminal.

La recuperación se realiza utilizando el programa LG Bridge, con la opción Recuperación de error de actualización. 

Vamos a ver el proceso paso a paso en los siguientes apartados.

Importante: antes de empezar...

Lee estos puntos antes de comenzar con el proceso:

  • La descarga del archivo de actualización del teléfono puede ocupar bastante espacio. Comprueba que tienes al menos 10 Gb libres en el ordenador.
  • Si durante el proceso de recuperación de emergencia se produce algún error, es muy probable que el terminal requiera de una intervención en el servicio técnico. Abre una solicitud de reparación.

Paso 1: descarga del programa LG Bridge Update

Puedes encontrar toda la información relevante a la descarga del programa en este enlace:

https://www.lg.com/es/posventa/microsites/movil/lg-bridge-instalacion

*si ya tienes instalado el programa LG Bridge, salta al siguiente paso

Paso 2: LG Bridge

Una vez descargado LG Bridge, procedemos a abrir y seleccionamos la pestaña: Actualización de software

Paso 3: activar el modo de emergencia del teléfono

Antes de empezar este paso asegúrate de que el terminal está apagado y no tiene conectado el cable USB. Una vez comprobado esto:

  • Conecta el cable USB al ordenador (pero no al terminal).
  • Pulsa y mantén pulsada la tecla VOL + (subir volumen) del teléfono.
  • Mientras mantienes pulsada la tecla VOL +, conecta el cable USB al terminal.
  • Tras unos instantes el dispositivo arrancará en modo de emergencia o download mode, suelta la tecla de volumen y espera unos instantes hasta que aparezca la pantalla del siguiente paso.

El teléfono ya ha arrancado en el modo de carga de emergencia. Dependiendo de tu modelo, la imagen puede cambiar.

Paso 4: iniciar la reparación de emergencia

1. En el programa LG Bridge, pulsa en el menú superior Actualización de software. Selecciona abajo a la derecha la opción Recuperación de error de actualización.

2. Una vez aceptadas las políticas, espera unos instantes a que LG Bridge revise el estado del dispositivo.

3. El programa te advertirá de que el proceso borra todos los datos del terminal (excepto la tarjeta SD). Marca la casilla y pulsa OK para comenzar el proceso.

4. Comenzará el proceso de actualización. Ten paciencia pues la descarga puede tardar bastante tiempo dependiendo del tamaño de la imagen de software. El teléfono puede reiniciarse durante el proceso, no desconectes el cable USB hasta que finalice.

Una vez el proceso finalice, el terminal se reiniciará para terminar con la actualización y posteriormente estará listo para utilizarse.
